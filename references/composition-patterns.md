# Composition Patterns

Choose one pattern per image.

## Claim Weighing

Use for: strong opinion, judgment, thesis, "this is true because...".

Drawing: a claim slip on a strange scale. Evidence bricks on one side; red unsupported fluff floating away. A worker adjusts the weight.

## Boundary Gate

Use for: trust, safety, permission, write gates, approvals, security, policy.

Drawing: a small gate or moat. Inputs arrive as envelopes. A worker checks stamps before letting anything cross.

## Trace Path

Use for: auditability, provenance, debugging, memory, reproducibility.

Drawing: a colored ribbon enters a black box and emerges with tags. A worker pulls the ribbon through inspection windows.

## Agent Handoff

Use for: multi-agent collaboration, delegation, queues, review loops.

Drawing: two or three workstations connected by a path. Each worker passes a sealed object, not a chat bubble.

## Before / After Evidence

Use for: messy-to-clean, guess-to-proof, manual-to-systematic.

Drawing: left side loose papers and leaking labels; right side sorted trays and sealed proof. Orange arrow in the middle.

## Failure Leak

Use for: bugs, brittle workflow, hallucination, silent failure, hidden cost.

Drawing: a pipe, box, or conveyor leaks small red slips. A worker patches only the root leak while other slips wait.

## Decision Brief

Use for: tradeoff, "Sun must decide", product decision, architecture choice.

Drawing: two paths with different toll booths or weights. A worker holds two stamped cards; unresolved decision remains visibly open.

## Moat Bridge

Use for: trust gradient, review-first workflows, safe automation.

Drawing: a narrow bridge over a white moat. Evidence bricks make the bridge solid. Unsupported claims cannot cross.

## Memory Ledger

Use for: Obsidian, logs, durable memory, knowledge systems.

Drawing: drawers or ledger shelves. Workers file only sealed cards; loose notes sit in an inbox tray.

## Metaphor Recipe

1. Convert the abstract idea into a physical operation: weighing, sorting, stamping, sealing, patching, replaying, guarding, bridging, routing.
2. Pick one low-tech object: scale, tray, gate, ribbon, pipe, ledger, drawer, cart, bridge, stamp, envelope.
3. Put the evidence worker inside the operation.
4. Add 3-6 labels that name the actual concepts from the article.
5. Remove anything that only explains the explanation.

## Anti-Repetition

Do not default to the same scale/gate/ribbon every time. If two shots in a list use similar metaphors, change one to a different physical operation.
